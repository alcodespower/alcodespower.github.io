wepy使用踩坑排雷

  1. 安装wepy
    npm install wepy-cli -g

  2. 生成项目
    wepy new <projectname>

  3. 实时编译
    wepy build --watch

有时候，它会说 please use wepy init standard demo ...这里我们只需要按他说的做，这样写：

$ wepy init standard <ProjectName>

这里的ProjectName是到时候文件夹的名字，
下面的除了ESLint是 “ n ”，其他直接回车就行了，这步骤插在第二三步之间。
