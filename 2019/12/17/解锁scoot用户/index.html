Oracle学习 Day01-安装

我个憨憨，手快，直接跳过了口令设置

如果在Oracle11g在安装过程中忘了进行口令配置，不要着急，无需卸载重载。可以通过命令行进行修改。

比如我们忘记解锁scott用户，可以先打开SQL plus工具。

输入用户：sys
输入口令：sys as sysdba

然后输入解锁scott用户语句（分号不要忘了）：alter user scott account unlock;

然后输入修改scott密码（分号不要忘了）：alter user scott identified by tiger;
